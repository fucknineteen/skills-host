# 第一章：小六壬三种变体对比

## 1.1 江氏小六壬（当前 skill 已覆盖）

**来源**：江春义《祖传小六壬占法精华总集》

### 核心规则
- **三维体系**：六宫（地）+ 六亲（人）+ 六神五星（天）
- **起课**：数字÷6取余/农历月日时/汉字笔画/抽签
- **排盘**：日宫→时宫→隔位排地支→定六亲→装六神→起五星
- **六宫**：大安(木)、留恋(火)、速喜(土)、赤口(金)、小吉(水)、空亡(土)
- **五星**：木星(天福·吉)→火星(天威·中)→土星(天武·中)→金星(天祸·凶)→水星(天贼·凶)→天空(天狱·空)
- **核心公式**：六亲受五星生助者吉，克者凶

### 与其他变体的核心差异
1. **三星体系五星体系**：江氏有完整的五星（木火土金水天空）断法，诸葛和民间版本无此系统
2. **六亲定位**：江氏以时辰地支为自身，按五行生克定六亲；民间版通常不设六亲
3. **隔位排地支**：江氏有时辰落宫隔位排地支的复杂步骤，其他变体通常简化

## 1.2 诸葛马前课（诸葛亮马前课）

**来源**：传为诸葛亮行军占卜所用，民间流传最广的简化版本
**文献出处**：《马前课》（敦煌遗书有类似占法记载）、民间手抄本

### 核心规则
- **掌诀定位**：六壬掌诀，在大安/留连/速喜/赤口/小吉/空亡六位上推算
- **起课**：**月日时三要素**，从寅位（大安）起正月，月上起日，日上起时
- **最简版本**：直接按月/日/时顺数，月→日→时三步到位

### 六掌诀含义（与江氏大同小异）
| 掌诀 | 五行 | 含义 | 数字 |
|------|------|------|------|
| 大安 | 木 | 光明正大、事事昌盛 | 一五七 |
| 留连 | 水 | 卒未归时、去者未回 | 二八十 |
| 速喜 | 火 | 喜事来临、行人有信 | 三六九 |
| 赤口 | 金 | 官非口舌、是非缠绕 | 四七十 |
| 小吉 | 木 | 人来喜时、凡事和合 | 一五七 |
| 空亡 | 土 | 音信稀时、谋事落空 | 三六九 |

**注意**：诸葛版本中留连为**水**，速喜为**火**，与江氏（留连火、速喜土）五行不同！

### 与江氏的核心差异
| 维度 | 江氏 | 诸葛马前课 | 民间简化版 |
|------|------|-----------|-----------|
| 五行体系 | 留连火/速喜土 | **留连水/速喜火** | 留连火/速喜火 |
| 五星系统 | 有（木火土金水天空） | **无** | 无 |
| 六亲断法 | 完整六亲生克 | **无** | 无 |
| 隔位排地支 | 有 | **无（直接顺数）** | 无 |
| 断法深度 | 三维（五星+六宫+六亲） | 单维（掌诀为主） | 单维 |
| 应期体系 | 有（快中慢+数字断） | 简化 | 极少 |
| 神煞 | 六神（青龙朱雀勾陈腾蛇白虎玄武） | **有时附会六神** | 极少 |
| 来源可靠性 | 江氏家传（有完整文本） | 托名诸葛亮（民间口传） | 民间口头传承 |

### 诸葛马前课特有的断法
- **失物**：大安去东方不远，留连在南方，速喜在南方或喜事人家，赤口在西方对门，小吉不出大门外，空亡寻不见
- **行人**：大安身未动，留连未回转，速喜在途中，赤口有口舌，小吉在路上，空亡无音信
- **婚姻**：大安和美，留连不顺，速喜媒人得力，赤口口舌，小吉和合，空亡不成

## 1.3 民间简化版小六壬

**特征**：最简化的掌诀占法，常见于民间算命摊和网络流传

### 与江氏的主要差异
1. **无六亲系统**：不断六亲生克，只看落宫吉凶
2. **无五星系统**：没有天福天祸天贼天空等五星照临
3. **简化起课**：仅用月日两柱，或仅用随机数字一柱
4. **数字对应简化**：1=大安, 2=留连, 3=速喜, 4=赤口, 5=小吉, 6=空亡（不取余数）
5. **应期模糊**：基本不推断应期

## 2.2 断法逻辑：体用生克（最核心）

### 体卦与用卦
- **体卦**：代表求测者本人（无动爻之卦）
- **用卦**：代表所问之事（有动爻之卦）
- **变卦**：动爻变化后形成的新卦（代表事情的结果/发展）
- **互卦**：本卦二三四爻和三四五爻组成（代表事情过程）

### 体用生克总纲
```
体克用 → 诸事吉（我能控制事）
用克体 → 诸事凶（事控制我）
体生用 → 有耗损（我生事，泄气）
用生体 → 有进益（事生我，得利）
体用比和 → 万事顺遂（五行相同，和谐）
```

### 五行生克与断事结合
| 事类 | 用卦宜 | 用卦忌 | 体卦宜 |
|------|--------|--------|--------|
| 求财 | 用生体（财来找我） | 体生用（我找财=赔） | 体克用（我控财） |
| 求官 | 用克体或体克用 | 体生用 | 官来克我为得官 |
| 婚姻 | 用生体/体用比和 | 用克体 | 阴阳得位 |
| 疾病 | 体克用（病可治） | 用克体（病加重） | 体生用=病缠绵 |
| 诉讼 | 体克用（我胜） | 用克体（对方胜） | 比和=和解 |
| 出行 | 用生体/体克用 | 用克体 | 体生用=出行破财 |
| 失物 | 体克用（可寻回） | 用克体（找不回） | 体生用=主动遗失 |
| 考试 | 用生体（考运好） | 用克体（不利） | - |

## 2.3 三要十应（高级断法）

### 三要（眼耳心）
- **眼要**：观其外应（起卦时看到的景象、人物、动作）
- **耳要**：听其声音（起卦时听到的声音、言语）
- **心要**：察其时义（起卦时的季节、时间、情境）

### 十应（十种外应）
| 应 | 内容 |
|-----|------|
| 天时 | 晴雨风云雪 |
| 地理 | 山川道路方位 |
| 人事 | 人物行为言语 |
| 时序 | 春夏秋冬昼夜 |
| 方卦 | 方位对应之卦 |
| 动物 | 飞禽走兽出现 |
| 静物 | 器物瓶罐 |
| 言语 | 旁边人说的话 |
| 声色 | 声音颜色 |
| 卦象 | 互卦变卦辅证 |

> **铁律**：外应克体卦=凶，外应生体卦=吉。外应优先于卦象本身。

## 2.4 常见格局与经典卦例

### 经典吉格
- **地天泰（坤上乾下）**：天地交泰，万事通顺
- **水火既济（坎上离下）**：事已成功，但需守成
- **风雷益（巽上震下）**：见善则迁，有利益
- **地泽临（坤上兑下）**：事物临近，即将到来

### 经典凶格
- **天地否（乾上坤下）**：天地不交，阻塞不通
- **火水未济（离上坎下）**：事未成功，需要时间
- **山雷颐（艮上震下）**：求人养己，等待时机
- **水雷屯（坎上震下）**：万事开头难

### 经典卦例：观梅占
邵雍观二雀争梅坠地，以年月日时起卦得「泽火革」变「泽山咸」：
- 兑为体=少女，用离克体兑金 → 少女受伤
- 变卦艮土生体兑金 → 伤不至死
- 断：明晚有少女折花，园丁驱逐，少女坠地伤股。**全部应验。**

## 3.2 刑冲合害详细断法

> 当前 skill 仅在《子平真诠》文件中提到刑冲合害的表面概念，但缺少**详细的断法逻辑**。

### 3.2.1 六冲（地支六冲）

```
子午冲 · 丑未冲 · 寅申冲 · 卯酉冲 · 辰戌冲 · 巳亥冲
```

### 冲的吉凶断法
| 冲的类型 | 含义 | 断法 |
|----------|------|------|
| 冲旺 | 旺者冲衰衰者拔 | 旺神冲衰神，衰神受损大 |
| 冲衰 | 衰者冲旺旺者发 | 衰神激旺神，旺神反而更旺 |
| 冲动不冲散 | 有合解冲/有会局解冲 | 减力但不破坏 |
| 冲出 | 冲墓库/冲开 | 冲开墓库，吉物出库则吉 |
| 冲夫妻宫 | 日支被冲 | 婚姻不稳、夫妻不和 |
| 冲父母宫 | 年支被冲 | 父母不睦或离家 |
| 冲事业宫 | 月支被冲 | 工作变动频繁 |

> **核心口诀**：冲就是变动。喜神被冲为凶，忌神被冲为吉。

### 3.2.2 六合（地支六合）

```
子丑合土 · 寅亥合木 · 卯戌合火 · 辰酉合金 · 巳申合水 · 午未合日月
```

### 合的吉凶断法
| 合的类型 | 含义 | 断法 |
|----------|------|------|
| 合化为喜 | 合化为用神 | 大吉 |
| 合化为忌 | 合化为忌神 | 凶 |
| 合走 | 用神被合走 | 凶（喜神被合去） |
| 合留 | 忌神被合走 | 吉（凶物被合制） |
| 争合 | 二合一天干 | 女命婚恋复杂 |
| 妒合 | 二合一支 | 人际关系复杂 |
| **天地鸳鸯合** | 天干五合+地支六合同时出现 | 婚恋/合作极强信号 |

### 3.2.3 三合局（最强合）

```
申子辰 → 水局  |  亥卯未 → 木局  |  寅午戌 → 火局  |  巳酉丑 → 金局
```

**三合局断法**：
- 三合局全 → 力量极大，改变五行格局
- 半合（申子/子辰/亥卯/卯未等） → 力量中等
- 三合局为用神 → 大吉，格局陡升
- 三合局为忌神 → 大凶，气势被夺

### 3.2.4 三刑

```
无礼之刑：子刑卯，卯刑子
恃势之刑：寅刑巳，巳刑申，申刑寅
无恩之刑：丑刑戌，戌刑未，未刑丑
自刑：辰刑辰，午刑午，酉刑酉，亥刑亥
```

### 刑的断法
| 刑 | 主事 | 断法 |
|-----|------|------|
| 子卯刑 | 无礼之刑 | 门户不宁、尊卑失序、色情之灾 |
| 寅巳申刑 | 恃势之刑 | 官非牢狱、车祸外伤、仗势欺人 |
| 丑戌未刑 | 无恩之刑 | 六亲寡恩、恩将仇报、同室操戈 |
| 自刑 | 自寻烦恼 | 自我伤害、自杀倾向、抑郁纠结 |

> **核心**：刑主动，逢刑必有事。刑在年柱→祖上；月柱→兄弟/事业；日柱→自己/配偶；时柱→子女。

### 3.2.5 六害（穿）

```
子未害 · 丑午害 · 寅巳害 · 卯辰害 · 申亥害 · 酉戌害
```

害不如冲刑猛烈，但代表暗中的妨害、小人、慢性损伤。

## 3.4 调候用神系统

> **当前 skill 仅提到"调候口诀"三句，缺少系统理论。** 调候源自《穷通宝鉴》（又名《栏江网》），是八字命理中决定格局高低的关键加分项。

**来源**：《穷通宝鉴》（明·佚名，清·余春台整理）、《造化元钥》

### 3.4.1 调候核心原理

调候解决的是**月令寒暖燥湿**对日主的影响：
- **金水生于冬令** → 太寒，需**火**暖局（丙火最佳）
- **木火生于夏令** → 太燥，需**水**润局（癸水最佳）
- **土生于辰戌丑未月** → 燥土需水，湿土需火

# 第四章：奇门遁甲基础

> **当前 skill 完全未覆盖。** 奇门遁甲被称为"帝王之术"，是三式（太乙、奇门、六壬）中最复杂的一门。

**来源**：《烟波钓叟赋》（宋·赵普）、《奇门遁甲秘笈大全》（明·刘伯温）、《遁甲演义》（明·程道生）

## 4.1 基本架构

### 时空四层模型
```
天时 → 九星（天盘）
地利 → 八门（人盘）
人和 → 八神（神盘）
格局 → 奇仪（地盘）
```

### 四层速查
| 层次 | 内容 | 作用 |
|------|------|------|
| **地盘** | 九宫八卦 + 天干（戊己庚辛壬癸+乙丙丁三奇） | 基础格局 |
| **天盘** | 九星（天蓬、天芮、天冲、天辅、天禽、天心、天柱、天任、天英） | 天时、先天运势 |
| **人盘** | 八门（休、生、伤、杜、景、死、惊、开） | 人事、出入吉凶 |
| **神盘** | 八神（值符、腾蛇、太阴、六合、白虎、玄武、九地、九天） | 神秘力量、暗局 |

### 核心参数
- 排盘依据：**年月日时四柱 + 节气（定阴阳遁）+ 用局数**
- 阳遁九局（冬至后）、阴遁九局（夏至后）

## 4.2 八门断法

八门是奇门遁甲**最实用的断事系统**。

| 门 | 五行 | 方位 | 吉凶 | 含义 | 用事 |
|----|------|------|------|------|------|
| **休门** | 水 | 北 | **吉** | 休息、休养、退隐 | 求财、婚姻、谒见贵人 |
| **生门** | 土 | 东北 | **大吉** | 生机、求财、生育 | 求财、种养、出行、嫁娶 |
| **伤门** | 木 | 东 | 凶 | 受伤、竞争、捕猎 | 渔猎、讨债（以毒攻毒） |
| **杜门** | 木 | 东南 | 平偏凶 | 堵塞、隐藏、技术 | 隐匿、闭门读书、技术攻关 |
| **景门** | 火 | 南 | 平 | 文书、信息、宴会 | 考试、谈判、献策 |
| **死门** | 土 | 西南 | **大凶** | 死亡、终结、固执 | 丧葬、狩猎、极少数用事 |
| **惊门** | 金 | 西 | 凶 | 惊恐、诉讼、口舌 | 捉贼、诉讼（本方占惊） |
| **开门** | 金 | 西北 | **大吉** | 开创、出行、上任 | 开业、远行、上任、搬迁 |

### 八门断事口诀
```
欲求财利往生方，埋葬须知死路强。
捕捉射猎伤门好，寻人须向景门方。
上官赴任开门吉，休门谒见贵人良。
杜门避难并绝迹，惊门捕盗最为良。
```

### 八门克应
- 门**生宫**为「和」→ 吉
- 宫**生门**为「义」→ 吉
- 门**克宫**为「迫」→ 凶
- 宫**克门**为「制」→ 平偏凶
