# 价格走势路径分类

## 分类逻辑

基于入场后1H K线数据，将价格走势分为8种路径类型：

| 路径类型 | 判定条件 |
|---------|---------|
| 横盘震荡 | 全程振幅 < 2% |
| 单边上涨 | 净涨>3%, 最大回撤<1%, 前后半段都涨 |
| 单边下跌 | 净跌<-3%, 最大反弹<1%, 前后半段都跌 |
| 先涨后跌 | 高点在前30%, 低点在后30%, 后半段跌幅过半 |
| 先跌后涨 | 低点在前30%, 高点在后30%, 后半段上涨 |
| 冲高回落 | 高点在前, 净跌<0, 后半段跌>2% |
| 探底回升 | 低点在前, 净涨>0, 后半段涨>2% |
| 宽幅震荡 | 振幅>5%, 净涨跌<2% |

## 实现

- `price_path_report.py` — 独立报告脚本，`--days N --coin COIN` 参数
- `review_path_enhancer.py` — 增强器，可写入 `reviews.json` 的 `price_path` 字段
- 分类函数 `classify_price_path(candles, entry_price)` 在两个脚本中共用

## 分类指标

每个路径分析输出：
- `net_change_pct`: 净涨跌
- `total_range_pct`: 振幅
- `global_high/low` + 出现时间
- `max_drawdown_pct`: 最大回撤
- `max_rally_pct`: 最大反弹
- `first_half/second_half`: 分段涨跌
