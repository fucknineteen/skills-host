# 第一章：什么是做T

## 1.1 基本概念

```
做T = 日内T+0操作 = 当天买卖同一标的，赚取差价

A股背景：
  股票是T+1（今天买，明天才能卖）
  但如果你已经持有这只股票 → 可以「今天卖昨天买的，今天再买回来」
  → 实现了变相的T+0
  → 这就是「做T」

BTC 永续合约：
  天然支持T+0（随时买随时卖）
  没有「T+1」的限制
  但这不意味着可以随便做——
  做T的核心不是「能买卖」，而是「在什么位置买卖」
```

## 1.2 做T的本质

```
做T ≠ 频繁交易
做T = 利用日内波动，降低持仓成本 或 增厚收益

核心目的：
  → 手里有仓位时 → 做T降成本
  → 手里没仓位时 → 做T赚波段

做T vs 趋势交易：
  趋势交易：今天买，拿3天，吃一段主升浪
  做T：      今天买，1小时后卖，吃日内的小波动
  
  趋势交易赚的是「方向的钱」
  做T赚的是「波动的钱」
```

## 1.3 BTC 永续合约做T的天然优势

```
A股做T的痛点          BTC永续合约的优势
─────────────────────────────────────
T+1限制               T+0，随时买卖
涨跌停板10%           无涨跌停
只能做多（正T）        可以做空（正T+反T）
手续费+印花税          仅手续费（0.05%）
4小时交易时间          24小时交易
单边市场              双向市场
```

# 第二章：做T的两种基本模式

## 2.1 正T（先买后卖）

```
正T = 低位买入 → 高位卖出 → 当天完成一轮

适用场景：趋势偏多，价格短期回调
前提条件：你认为价格还会涨回来
```

### 正T流程

```
你持有 0.1 BTC 空单（-0.1 BTC）

14:00 价格 $73,500 → 你判断短期会反弹
      → 买入 0.1 BTC（平掉空单）→ 持仓归零
15:00 价格 $73,800 → 反弹了$300
      → 卖出 0.1 BTC（重新开空）→ 持仓恢复 -0.1 BTC

结果：
  空单成本从 $73,500 降到了 $73,800
  每张合约赚了 $300 的差价
  持仓量不变，方向不变
```

### 正T的另一种形式（做多T）

```
趋势偏多，你持有多单

13:00 价格 $73,200 → 补仓做多 0.1 BTC（持仓+0.2 BTC）
14:30 价格 $73,600 → 卖出 0.1 BTC（持仓回到+0.1 BTC）

结果：
  没有改变持仓量
  但赚了 $400 的差价
  你的持仓成本被摊低了
```

## 2.2 反T（先卖后买）

```
反T = 高位卖出 → 低位买回 → 当天完成一轮

适用场景：趋势偏空，价格短期反弹
前提条件：你手里有仓位可以卖
```

### 反T流程

```
你持有 0.1 BTC 空单（做空 -0.1 BTC）

10:00 价格 $73,200 → 你判断短期会继续跌
      → 加仓做空 0.1 BTC（持仓变为 -0.2 BTC）
11:30 价格 $72,800 → 跌了$400
      → 平掉 0.1 BTC 空单（持仓回到 -0.1 BTC）

结果：
  空单仓位不变
  但赚了 $400 差价
```

## 2.3 正T vs 反T 的选择

```
选择正T还是反T，取决于：

1. 你当前的持仓方向
   持多单 → 正T（回调补多，反弹出）
   持空单 → 反T（反弹补空，下跌出）

2. 威科夫阶段
   Accumulation/Re-accumulation → 正T为主（逢低买）
   Distribution/Re-distribution → 反T为主（逢高卖）

3. VWAP 位置
   价格在 VWAP 上方 → 正T（回调到 VWAP 时买）
   价格在 VWAP 下方 → 反T（反弹到 VWAP 时卖）

4. 成交量分布
   价格在 POC 下方 + 接近下方 HVN → 正T
   价格在 POC 上方 + 接近上方 HVN → 反T
```

# 第三章：如何识别做T机会

## 3.1 做T的黄金窗口

不是所有时间都适合做T。以下是最佳时段：

```
BTC 日内波动最大的时段（BJT）：

08:00-10:00  亚洲开盘，波动率 ↑↑    ← 最佳做T窗口
15:00-17:00  欧洲开盘，波动率 ↑↑    ← 最佳做T窗口
20:00-23:00  美股开盘+欧美重叠       ← 最佳做T窗口
02:00-06:00  低波动期                 ← 不适合做T
```

### 如何判断「今天适合做T」

```
适合做T的日子：
  ✓ 价格在 HVN 之间有明确摆动空间
  ✓ VWAP ±2σ 带宽 > 0.5%（有足够波动空间）
  ✓ Bollinger Band 带宽在扩张
  ✓ ADX 在 15-25 之间（不是强趋势）
  ✓ 成交量分布呈单峰或双峰（不是强趋势）

不适合做T的日子：
  ✗ 强趋势日（ADX > 30）→ 做T容易被趋势碾压
  ✗ 重大事件前（FOMC/CPI）→ 波动不可控
  ✗ VWAP ±2σ 带宽 < 0.3% → 没有足够差价空间
  ✗ 凌晨 02:00-06:00 → 波动太小
```

## 3.2 做T的信号组合

### 组合一：VWAP 回归（最常用）

```
信号：
  价格偏离 VWAP 到 ±1.5σ → 等待回归 VWAP

做多T（正T）：
  价格跌到 VWAP -1.5σ → 补仓
  价格回到 VWAP → 抛出补仓部分
  差价 = 约 1.5σ（通常 $200-400）

做空T（反T）：
  价格涨到 VWAP +1.5σ → 加仓空
  价格回到 VWAP → 平掉加仓部分
  差价 = 约 1.5σ
```

### 组合二：HVN 之间摆动

```
信号：
  价格在两个 HVN 之间规律摆动

例如：
  下方 HVN = $73,200（支撑）
  上方 HVN = $73,800（阻力）

正T：
  价格跌到 $73,200-73,300 → 5m 出现锤子线 → 补仓
  价格涨到 $73,700-73,800 → 5m 出现流星线 → 抛出

差价 = $400-600/轮
一天可以做 2-3 轮
```

### 组合三：FVG 回踩

```
信号：
  15m 级别出现未回补的 FVG

正T（看涨 FVG）：
  价格回踩到 FVG 区间 → 补仓
  价格反弹到前高 → 抛出
  差价取决于 FVG 到前高的距离

反T（看跌 FVG）：
  价格反弹到 FVG 区间 → 加空
  价格回落到前低 → 平掉
```

## 3.3 信号优先级

```
做T信号优先级：

⭐⭐⭐⭐⭐  VWAP ±2σ 回归 + 缩量（最强）
⭐⭐⭐⭐   HVN → HVN 摆动 + 缩量测试
⭐⭐⭐⭐   15m FVG 回踩 + 5m 反转K线
⭐⭐⭐     POC 回归
⭐⭐       前日 VAH/VAL 回归
  
少于 ⭐⭐⭐ 的信号 → 不做T
```

# 第四章：做T的入场与出场

## 4.1 入场规则

```
做T入场三要素（缺一不可）：

1. 位置到了
   → 价格在目标区间内（VWAP ±2σ / HVN / FVG 区间）

2. 量确认了
   → 缩量触及支撑（正T）/ 缩量触及阻力（反T）
   → 或者：放量突破后回踩

3. 形态出来了
   → 5m 级别出现反转K线：
   → 正T：锤子线 / 看涨吞没 / Morning Star
   → 反T：流星线 / 看跌吞没 / Evening Star

只有位置到了 ≠ 可以入场
位置 + 量确认 + 形态 = 三要素齐全才入场
```

## 4.2 出场规则

```
做T出场的三个「必须」：

1. 必须提前设好止盈
   → 到目标位就走，不贪
   → 目标位 = VWAP / 下一个 HVN / FVG 另一侧

2. 必须设止损
   → 做T ≠ 不怕亏
   → SL = 入场位反方向 0.15-0.25%

3. 必须当天（或该时段内）完成
   → 做T不能变成「补仓被套」
   → 如果价格没有按预期走到目标位
   → 到收盘（或该时段结束）必须平掉
   → 宁愿微亏，不把做T变成加仓
```

### 做T失败的处理

```
做T最忌讳：补仓后价格继续跌 → 不止损 → 变成重仓被套

铁律：
  ▸ 做T单的 SL 必须独立于主仓位的 SL
  ▸ 做T单亏损不能超过当日做T总利润的 50%
  ▸ 连续3次做T止损 → 今天不适合做T → 停手
  
做T失败的标准处理：
  到了 SL → 立刻砍掉 → 不要想「再等等」
  砍掉后 → 不要立刻反手 → 冷静 15 分钟
  15 分钟后 → 重新评估是否还有做T机会
```

# 第五章：做T的仓位与风控

## 5.1 做T仓位大小

```
做T仓位 ≠ 主仓位
做T仓位 = 主仓位的 20-50%

如果你的主仓位是 0.1 BTC：
  做T仓位 = 0.02-0.05 BTC
  绝对不要用全部仓位做T

为什么？
  → 做T是「锦上添花」，不是「主要盈利来源」
  → 用大仓位做T → 心态会变 → 操作会变形
  → 做T失败被套 → 小仓位不致命 → 大仓位可能毁掉整个交易计划
```

### 仓位计算表

| 主仓位（BTC） | 保守做T仓位 | 激进做T仓位 |
|-------------|-----------|-----------|
| 0.05 | 0.01 | 0.02 |
| 0.10 | 0.02 | 0.05 |
| 0.20 | 0.05 | 0.10 |
| 0.50 | 0.10 | 0.25 |

## 5.2 做T的盈亏比

```
做T的盈亏比通常较小（1:0.8 ~ 1:1.5）
这不同于趋势交易（1:2 ~ 1:5）

为什么做T可以接受较低的盈亏比？
  → 做T胜率高（70-80% vs 趋势交易的 40-50%）
  → 做T频率高（一天 2-5 次 vs 趋势交易几天一次）
  → 高胜率 × 适当频率 × 小额利润 = 稳定收益

但如果做T的胜率低于 55% → 停下来反思
你大概率在「强趋势中逆势做T」
```

## 5.3 做T的每日限额

```
做T的每日风控：

每日做T次数上限：5 次
每日做T总亏损上限：本金的 2%
单笔做T亏损上限：本金的 0.5%

如果达到亏损上限 → 今天停止做T
如果连续三天亏损 → 本周停止做T，复盘原因
```

# 第六章：BTC 永续合约做T实战

## 6.1 案例一：VWAP 回归做T（经典震荡日）

**场景**：BTC 在 $73,000-73,800 震荡，ADX=20（非趋势日）

```
主仓位：空单 -0.1 BTC @ $73,500
做T仓位：0.03 BTC（30% 主仓）

当日 VWAP（08:00 重置）：
  09:00 VWAP = $73,300
  11:00 VWAP = $73,400
  13:00 VWAP = $73,350

做T #1（正T）：
  13:30 价格跌到 $73,080（VWAP 下带1 = $73,100）
  13:35 5m 出现锤子线 + 缩量 → 入场做多 0.03 BTC
  入场：$73,100 | SL：$72,950（-0.2%）
  
  14:15 价格回到 $73,380（= VWAP）→ 卖出 0.03 BTC
  盈利：$280（0.03 BTC × $280 = $8.4）

做T #2（反T）：
  15:00 价格反弹到 $73,650（前 HVN 上沿）
  15:05 5m 出现流星线 + 缩量 → 加空 0.03 BTC
  入场：$73,650 | SL：$73,820
  
  16:20 价格回落到 $73,350（POC）→ 平掉 0.03 BTC
  盈利：$300（0.03 BTC × $300 = $9.0）

当日做T总结：
  做T #1：+$8.4
  做T #2：+$9.0
  主仓位浮动盈亏：（未触发 TP/SL）
  做T 贡献：$17.4（占本金的 17.4% 收益增强）

关键点：
  → 两次做T都基于 VWAP/HVN 位置
  → 每次都有 5m 反转K线确认
  → 做T仓位 30%，不影响主仓位
```

## 6.2 案例二：做T失败 → 止损 → 停手

**场景**：BTC 在下跌趋势中，ADX=32（强趋势）

```
主仓位：空单 -0.1 BTC @ $74,000
做T仓位：0.03 BTC

14:00 价格跌到 $72,800 → 你判断短期见底
      → 正T做多 0.03 BTC（平掉部分空单）
      
14:30 价格没有反弹，继续跌到 $72,500
      → 你加仓做多 0.03 BTC（又平掉一些空单）
      → 累计平掉 0.06 BTC 空单
      → 心态开始变：「已经跌这么多了，该反弹了吧」

15:00 价格跌到 $72,200
      → SL 没设！或 SL 太宽！
      → 浮亏 $600（0.06 BTC × $1,000 = $60）

15:30 价格跌到 $71,800
      → 心态崩溃，全部平仓
      → 做T亏损：$60

更糟的是：
  主仓位从 -0.1 BTC 被减到了 -0.04 BTC
  错过了 $74,000 → $71,800 的 $2,200 跌幅
  损失的利润：0.06 BTC × $2,200 = $132

教训：
  ✗ 在强趋势中逆势做T = 灾难
  ✗ 没有设 SL
  ✗ 亏损后加仓（摊平成本）→ 错上加错
  ✗ 没有在 14:30 认识到「今天不适合做T」
  
正确做法：
  14:30 止损出来 → 亏损 $15（0.03 BTC × $500）
  承认错误 → 今天不做T了 → 保住主仓位
```

# 第七章：做T心态与常见错误

## 7.1 做T心态铁律

```
1. 做T是「锦上添花」，不是「雪中送炭」
   赚趋势的钱是主食，做T是甜点
   不要为了做T而影响主仓位的判断

2. 宁可错过，不要做错
   信号不明确 → 不做
   趋势太强 → 不做
   波动太小 → 不做
   心态不好 → 不做

3. 做T亏损不可怕，可怕的是：
   ▸ 亏损后加仓（摊平成本）
   ▸ 不止损，让做T变成加仓
   ▸ 频繁交易，手续费吃掉利润
   ▸ 情绪化操作

4. 做T的最高境界：
   不是「每天都能做」，而是「知道今天不能做」
```

## 7.2 常见错误清单

```
错误1：趋势市逆势做T
  症状：ADX > 30，还想着「涨多了该跌了」
  治疗：趋势市只做顺势T（趋势回调时入场）

错误2：不设止损
  症状：「再等等就回来了」
  治疗：入场的同时必须设 SL

错误3：仓位越做越大
  症状：第一笔 0.03 → 第二笔 0.05 → 第三笔 0.1
  治疗：做T仓位固定，不因盈亏调整

错误4：做T做成趋势单
  症状：「赚了$500，再看看能不能赚$1000」
  治疗：到目标位必须走，这是T不是趋势

错误5：频繁做T
  症状：一天 10+ 次，手续费 $50+，利润 $30
  治疗：手续费 0.05% × 2 = 0.1%/次，10 次 = 1% 本金
        净赚不够手续费的 → 少做

错误6：在低波动时段做T
  症状：凌晨 2-6 点，波动 $20-30，也去做T
  治疗：波动 $20，手续费 $5 → 不够滑点的 → 不做
```
