# 第一章：什么是 VWAP

## 1.1 基本定义

**VWAP（Volume Weighted Average Price）** = 成交量加权平均价格。它不是简单的 (H+L+C)/3，而是**每笔成交价按其成交量加权**后的均价。

```
简单理解：
  如果你买了 100 BTC，其中：
  - 60 BTC 在 $50,000 买入
  - 40 BTC 在 $52,000 买入
  
  VWAP = (60 × 50000 + 40 × 52000) / 100 = $50,800
  普通均价 = (50000 + 52000) / 2 = $51,000 ← 不准确！
```

## 1.2 为什么 VWAP 重要

VWAP 是**机构交易的核心基准**：

1. **算法执行基准**：大型机构用 VWAP 算法拆分大单，目标是「以不差于 VWAP 的价格成交」
2. **被动资金的锚**：ETF、指数基金在再平衡时参考 VWAP
3. **日内趋势的「公平价值」**：VWAP 线代表了当天所有参与者的平均成本
4. **自我实现的预言**：太多算法盯着 VWAP，导致价格在 VWAP 附近有真实的支撑/阻力效应

## 1.3 VWAP vs 简单移动平均线

| 特性 | SMA / EMA | VWAP |
|------|----------|------|
| 计算方式 | 仅价格 | 价格 × 成交量 |
| 重置时间 | 连续滚动 | 每日重置（标准VWAP） |
| 反映信息 | 纯价格趋势 | 参与者平均成本 |
| 机构使用 | 有限 | 核心基准 |
| 支撑/阻力质量 | 中等 | 高（尤其在日内） |
| 背离分析 | 可以 | 可以，且更有意义 |

# 第二章：VWAP 的数学原理

## 2.1 标准 VWAP 公式

```
VWAP = Σ(典型价格 × 成交量) / Σ成交量

其中：
  典型价格 = (High + Low + Close) / 3

每日重置，从当天第一根K线开始累加。
```

### 计算示例

```
K线1: H=100, L=98, C=99, V=1000
  典型价 = (100+98+99)/3 = 99
  累积PV = 99 × 1000 = 99,000
  累积V = 1000
  VWAP = 99,000/1000 = 99

K线2: H=102, L=99, C=101, V=800
  典型价 = (102+99+101)/3 = 100.67
  累积PV = 99,000 + 100.67×800 = 179,536
  累积V = 1000 + 800 = 1800
  VWAP = 179,536/1800 = 99.74
```

## 2.2 关键特性

```
特性1：VWAP 永远不会「跳跃」
  → 每根新K线只微调累计均价，VWAP线始终平滑

特性2：上午的成交量权重 > 下午
  → 因为上午累积成交量少，每笔新成交对VWAP影响大

特性3：收盘时 VWAP = 当天的真实平均成交价
  → 这是VWAP最有意义的时刻

特性4：VWAP 是单日指标
  → 标准VWAP在UTC 00:00重置（传统市场）
  → BTC 没有「收盘」，通常以 00:00 UTC (08:00 BJT) 重置
```

## 2.3 BTC 特有的 VWAP 重置问题

加密货币 24 小时交易，没有「开盘」和「收盘」。VWAP 重置时间的选择影响策略：

| 重置时间 | 对应 | 适用场景 |
|---------|------|---------|
| **00:00 UTC（08:00 BJT）** | 日线换线 | ⭐ 推荐，与OKX日线K线对齐 |
| **16:00 UTC（00:00 BJT）** | 北京时间午夜 | 隔夜交易参考 |
| **13:30 UTC（21:30 BJT）** | 美股开盘 | 关注美股联动时段 ⭐ |
| **不重置（连续VWAP）** | — | 需要锚定VWAP替代 |

**推荐**：对 BTC 使用两个 VWAP：
- 主 VWAP：重置于 08:00 BJT（对标 OKX 日线）
- 辅助 VWAP：重置于 21:30 BJT（美股开盘）

# 第三章：标准 VWAP vs 锚定 VWAP

## 3.1 标准 VWAP（Session VWAP）

每天重置的 VWAP，从当天第一根K线开始计算。

```
用途：
  ✓ 日内趋势判断（价格在VWAP上方=日内偏多）
  ✓ 日内超买超卖（偏离VWAP多远算极端？）
  ✓ 算法交易执行基准
  ✗ 不适用于多日持仓参考
  ✗ 不适合判断周级别的「公平价值」
```

## 3.2 锚定 VWAP（Anchored VWAP）

从**用户指定的某个重要事件点**开始计算，永不重置（或直到下一个锚点）。

### 常用锚点

| 锚点类型 | 从何处开始 | 用途 |
|---------|-----------|------|
| **趋势启动点** | 威科夫 Spring / SOS 的起点 K线 | 衡量整个趋势的平均参与成本 |
| **重大事件** | FOMC 公布、ETF 获批、CPI 发布 | 事件后的资金成本 |
| **波段高点/低点** | 重要的道氏高点或低点 | 判断是否回踩了「合理」的价格区 |
| **持仓开仓点** | 你自己的开仓K线 | 你的成本 vs 市场平均成本 |

### 锚定 VWAP 的计算

```
与标准VWAP相同，只是起始K线不是「今天第一根」，而是你指定的锚点K线。

例如：
  从 2026-05-20 08:00 BTC低点 $72,600 开始计算锚定VWAP
  → 此后每根新K线都参与累加
  → 这个VWAP从不重置，直到你手动更换锚点
```

### 锚定 VWAP 的独特价值

```
锚定 VWAP = 从关键事件以来的「市场平均持仓成本」

如果你的成本 < 锚定VWAP → 你比市场平均成本低 → 优势
如果你的成本 > 锚定VWAP → 你比市场平均成本高 → 劣势

当价格回踩锚定VWAP时：
  → 市场整体不赚不赔
  → 这是「公平」的价格
  → 趋势延续的关键测试点
```

# 第四章：VWAP 标准偏差带

## 4.1 什么是 VWAP 带

VWAP 带 = VWAP ± n × 标准偏差（基于典型价格）

```
VWAP上带1 = VWAP + 1 × σ
VWAP上带2 = VWAP + 2 × σ
VWAP上带3 = VWAP + 3 × σ

VWAP下带1 = VWAP - 1 × σ
VWAP下带2 = VWAP - 2 × σ
VWAP下带3 = VWAP - 3 × σ
```

## 4.2 标准偏差（σ）的计算

```
σ = √(Σ(典型价 - VWAP)² × 成交量 / Σ成交量)

简化理解：
  如果典型价与VWAP的偏离大、且量大 → σ变大 → 带宽变宽
  如果典型价紧贴VWAP、量小 → σ变小 → 带宽变窄
```

## 4.3 各带的意义

| 带 | 理论概率（正态分布） | 实际含义 | 交易信号 |
|----|---------------------|---------|---------|
| ±1σ | 68.3% | 正常波动区 | 无信号，价格正常 |
| ±2σ | 95.4% | 延伸波动区 | 短线回归机会 |
| ±3σ | 99.7% | 极端波动区 | **强回归信号**，罕见 |

### 实战规则

```
价格在上带1和上带2之间 → 偏强，但未极端，趋势可延续
价格突破上带2 → 短线超买，但强趋势可能沿上带2运行
价格突破上带3 → 极端超买，高概率回归VWAP
价格在下带1和下带2之间 → 偏弱，但未极端
价格突破下带2 → 短线超卖
价格突破下带3 → 极端超卖，高概率回归VWAP
```

## 4.4 BTC 特化带宽

BTC 波动率高于传统资产。在实战中，常使用以下调整：

```
保守带宽：±1.5σ, ±2.5σ, ±3.5σ（给更多空间）
标准带宽：±1σ, ±2σ, ±3σ（标准公式）
激进带宽：±0.8σ, ±1.6σ, ±2.4σ（更频繁的回归信号）
```

**推荐**：BTC 1H 级别用标准带宽，5m/15m 用保守带宽。

# 第五章：VWAP 作为动态支撑/阻力

## 5.1 VWAP 的引力和斥力

```
引力效应（均值回归）：
  - 当价格大幅偏离VWAP时，有回归倾向
  - ±2σ 以上 → 70-80%概率回归到±1σ内
  - ±3σ 以上 → 85-95%概率回归

斥力效应（趋势）：
  - 当价格紧贴VWAP但反复不穿越时，VWAP成为支撑/阻力
  - 几次测试VWAP后弹开 → VWAP防线确认
```

## 5.2 VWAP 测试的质量评估

| 测试特征 | 质量 | 含义 |
|---------|------|------|
| 缩量触及VWAP后立即弹开 | ⭐⭐⭐⭐⭐ | 极强支撑/阻力，趋势健康 |
| 平量触及VWAP，小幅穿透后弹回 | ⭐⭐⭐ | 支撑/阻力有效但有消耗 |
| 放量触及VWAP并收盘在上方（做空）/下方（做多） | ⭐ | VWAP防线面临突破风险 |
| 反复穿越VWAP | 无意义 | VWAP不再提供有效参考 |

### 高质量 VWAP 测试的特征

```
1. 触及速度：快速冲向VWAP后立即弹开（不是缓慢接近）
2. 成交量：触及瞬间量缩小（而不是放量）
3. 影线：仅影线触及，实体未收在VWAP另一侧
4. K线形态：触及后形成反转形态（锤子线/吞没/doji）
5. 次数：第1-2次测试比第5-6次更可靠
```

## 5.3 多 VWAP 汇聚区

当标准 VWAP 和锚定 VWAP 在同一价格区域汇聚时，该区域的支撑/阻力强度翻倍。

```
汇聚信号 = 三重确认：
  1. 标准VWAP（当天参与者的平均成本）
  2. 锚定VWAP（趋势启动以来的平均成本）
  3. 成交量分布POC（见第七章）
  
  三个指标指向同一价格区 → 极强支撑/阻力
```

# 第六章：VWAP 交易策略

## 6.1 策略一：VWAP 均值回归（震荡市）

```
条件：
  - 趋势不明确（ADX < 20 或 价格在区间震荡）
  - 价格突破VWAP上带2或下带2

做多规则：
  1. 价格触及 VWAP下带2（±2σ下方）
  2. 5m K线出现看涨吞没或锤子线
  3. 入场：确认K线收盘后
  4. SL：下带3下方
  5. TP：VWAP线 或 上带1
  6. 盈亏比：通常 1:1.5 ~ 1:2

做空规则（镜像）：
  1. 价格触及 VWAP上带2
  2. 5m K线出现看跌吞没或流星线
  3. SL：上带3上方
  4. TP：VWAP线 或 下带1
```

**注意**：这种策略在强趋势中会持续亏损（趋势会沿着带运行）。

## 6.2 策略二：VWAP 趋势跟随（趋势市）

```
条件：
  - 趋势明确（价格持续在VWAP同一侧）
  - 价格回踩VWAP但不穿越

做多规则：
  1. 价格始终在VWAP上方（趋势偏多）
  2. 价格缩量回踩VWAP
  3. 5m/15m 出现看涨反转信号
  4. 入场：VWAP线上方稍高位置（防止假穿越）
  5. SL：VWAP线下方 0.15-0.25%
  6. TP1：上带1（1:1~1:1.5）
  7. TP2：上带2（1:2~1:3）

做空规则（镜像）：
  1. 价格始终在VWAP下方
  2. 价格缩量反弹至VWAP
  3. 入场：VWAP线下方
  4. SL：VWAP线上方 0.15-0.25%
```

## 6.3 策略三：VWAP 突破（趋势启动确认）

```
条件：
  - 价格在VWAP一侧盘整后突破

做多突破规则：
  1. 价格在VWAP下方盘整数小时
  2. 突然放量上穿VWAP（突破K线量>1.5倍均量）
  3. 入场：突破K线收盘确认后
  4. SL：突破前低点 或 VWAP线下方0.2%
  5. TP1：上带1
  6. TP2：上带2

关键：突破VWAP的K线必须放量，缩量突破VWAP通常是假突破。
```

## 6.4 策略四：VWAP 背离交易

```
价格与VWAP的背离有两种：

A. 价格创新高，但偏离VWAP的幅度在减小
   → 上涨动能衰减，可能反转
   
   例如：第一波冲到+2.5σ，回调后第二波只到+1.8σ
        → VWAP带背离 → 动能衰竭 → 准备做空

B. 价格创新低，但偏离VWAP的幅度在减小
   → 下跌动能衰减，可能反转

VWAP背离比传统RSI/MACD背离更可靠，因为它融入了成交量。
```

# 第七章：VWAP + 成交量分布

## 7.1 两者的互补关系

| 维度 | VWAP | 成交量分布 |
|------|------|-----------|
| 数据基础 | 加权平均线 | 价格-成交量直方图 |
| 时间维度 | 日内动态 | 静态（固定区间）或动态（滚动） |
| 回答的问题 | 「今天平均成本是多少？」 | 「哪个价格成交量最集中？」 |
| 形态 | 一条线 | 水平分布图 |

## 7.2 VWAP + 成交量分布的组合信号

### 最强信号：VWAP = POC

```
当 VWAP 线与成交量分布的 POC（Point of Control / 最大成交量价位）重合时：
  → 这是当天「最公平的价格」
  → 多空双方在此达成最大共识
  → 极强的支撑/阻力
```

### 次强信号：VWAP 在 VA 内

```
VWAP 落在成交量分布的价值区（Value Area = 70%成交量区间）内：
  → VWAP是「合理的」
  → VWAP的支撑/阻力效应正常
```

### 弱信号：VWAP 在 VA 外

```
VWAP 落在价值区外：
  → 当天价格分布异常
  → VWAP 可能不反映真正的「公平价值」
  → VWAP 的参考价值降低
```

## 7.3 实战应用

```
步骤1：加载当天的成交量分布
步骤2：叠加VWAP线 + ±1σ/±2σ带
步骤3：观察VWAP与POC的位置关系

如果 VWAP ≈ POC → VWAP策略高置信
如果 VWAP 远 ≠ POC → 以成交量分布的VA为优先参考
```

# 第八章：BTC 永续合约实战案例

## 8.1 案例一：VWAP 均值回归（震荡日）

**场景**：BTC 在 $72,500-73,500 区间盘整，ADX=18

```
08:00 VWAP 重置，初始 VWAP = $72,950
10:30 价格跌至 $72,550（VWAP下带1 = $72,600）
11:15 价格触及 $72,450（VWAP下带2 = $72,480）
      5m出现锤子线 + 缩量 → 入场做多
      
入场：$72,480 | SL：$72,350（下带3下方）
TP1：$72,950（VWAP线）→ 14:00 触及 ✅ +$470
TP2：$73,350（上带1）→ 触及一半仓位

结果：两段获利，TP1 + TP2 合计约 $650/张
```

## 8.2 案例二：VWAP 趋势跟随（强趋势日）

**场景**：BTC ETF 大额流入消息公布，从 $71,000 启动

```
VWAP（08:00重置）：
  09:00 VWAP = $71,200
  10:00 VWAP = $71,500
  11:00 VWAP = $71,800
  12:00 VWAP = $72,200
  13:00 价格回踩 $72,200 → 等于VWAP！

分析：
  - 价格全天在VWAP上方 → 趋势明确偏多
  - 回踩VWAP缩量（量=0.7x均量） → 健康回调
  - 5m在VWAP处出现看涨吞没 → 入场信号

入场：$72,220 | SL：$72,050（VWAP下方0.23%）| 杠杆：50x | 保证金：8U
TP1：$72,600（上带1）→ 15:00触及 ✅
TP2：$73,000（上带2）→ 18:00触及 ✅

盈亏：TP1 +$380, TP2 +$780
```

## 8.3 案例三：VWAP 假突破陷阱

**场景**：价格在VWAP上方，突然放量下穿VWAP，但...

```
14:00：BTC从$73,500放量下穿VWAP（$73,200）
       看起来是强势看空信号
       
但是：
  14:05：下穿后立即弹回VWAP上方
  14:10：在VWAP上方形成看涨吞没
  14:15：VWAP再次被确认支撑

这是经典的 VWAP 假突破（Fake Breakdown）：
  → 放量下穿引诱空头
  → 迅速收复 → 空头被套
  → 后续加速上涨（空头回补 + 多头追入）

教训：
  ▸ 单根K线突破VWAP不可靠
  ▸ 需要等待**收盘确认** — K线实体必须收在VWAP另一侧
  ▸ 假突破后回到原方向，是最佳入场机会
```

# 第九章：VWAP 在现有分析体系中的位置

## 9.1 与两级分工的整合

| 层级 | VWAP 的用法 |
|------|-----------|
| 🧭 趋势判断 | VWAP 斜率 + 价格在VWAP哪一侧 = 日内趋势方向 |
| 🎯 执行定位 | VWAP ± 2σ 回归 + VWAP线回踩 = 精确入场点 |

## 9.2 VWAP 与其他方法的协同

### VWAP + 威科夫

```
威科夫 Phase C（Spring位置）如果正好在 VWAP下带2：
  → 双重确认了这是一个「极端抛售后的弹簧」位置
  → 高概率反转

威科夫 Phase B（UTAD位置）如果正好在 VWAP上带2：
  → 双重确认了这是一个「极端拉升后的陷阱」位置
  → 高概率反转
```

### VWAP + 道氏理论

```
上升趋势中的 HL（Higher Low）如果在 VWAP 线附近形成：

  → VWAP 阻力确认了 LH 的有效性
```
