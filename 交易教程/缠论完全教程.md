## 1. 起源与哲学基础

### 1.1 缠中说禅其人

**缠中说禅**（本名据传为李彪，亦有争议）是中国互联网早期（约2002-2008年）最具传奇色彩的民间思想家、投资家之一。他以"缠中说禅"为网名，在"天涯社区"、"强国论坛"等平台发表了大量涵盖哲学、佛学、诗词、中医、数学、经济学以及股票技术分析的雄文。

缠中说禅将自己在金融市场十余年的实战经验，结合中国传统哲学思想与现代数学逻辑，创立了一套**自洽的、完整的走势分析体系**，后世称之为"缠论"（Chan Theory）。

> **核心追求**：缠论的目标不是"预测市场"，而是**完全分类**——对任何走势的任何时刻，都能给出当下唯一的、确定性的分类，从而指导操作。

### 1.2 缠论的哲学基础

缠论的哲学根基融合了三大来源：

| 来源 | 核心思想 | 在缠论中的体现 |
|------|----------|----------------|
| **佛学（禅宗）** | 当下即是、不昧因果 | 只对"当下"做分类，不预测未来 |
| **道家** | 阴阳相生、物极必反 | 多空转化、走势的生住坏灭 |
| **现代数学** | 集合论、递归、拓扑 | 走势的递归结构、级别间的自相似性 |

缠中说禅反复强调：**"市场无需预测，只需完全分类"**。这意味着——

- 你不需要知道明天涨还是跌
- 你需要知道的是：在每一种可能的情况下，你应该怎么做
- 这就是"不测而测"的智慧

### 1.3 "走势终完美"——第一原理

> **走势终完美**：任何级别的任何走势类型，最终都要完成。

这是缠论的第一原理，是所有定理和推导的出发点。它有几层含义：

1. **任何走势类型必然终结**：一个上涨趋势不可能永远涨下去，它必将结束并转化为盘整或下跌
2. **任何未完成的走势必然继续完成**：一个正在构建中枢的盘整，中枢必然要完成（走出离开段）
3. **级别的递归保证**：低级别走势的不断生灭，构成了高级别走势的演化

**直观理解**：

```
一个上涨趋势 = 至少两个同向中枢 + 连接段
┌─────────────────────────────────┐
│  进入段 ↗ 中枢1 → 连接段 ↗ 中枢2 → 离开段 ↗  │
└─────────────────────────────────┘
当离开段相对于进入段发生背驰时，趋势"完美"（完成）
```

"走势终完美"是缠论最具革命性的洞见：它相当于说，走势永远在"从一种类型向另一种类型转化"——盘整要么延续，要么转化为趋势；趋势要么延续（延伸），要么转化为更大级别的盘整或反趋势。**转化完成的那一刻，就是"完美"的那一刻。**

## 2. 分型（顶分型/底分型）

### 2.1 什么是分型

**分型（Fractal）** 是缠论最基础的构建单元，由连续3根K线组成。

#### 顶分型（Top Fractal）

三根K线中，**中间K线的高点是三根中最高的，中间K线的低点也是三根中最高的**。

```
顶分型示意：
      ██  ← 第2根（最高）
     /  \
    /    \
 ███      ██
第1根    第3根

条件：H2 > H1 且 H2 > H3，且 L2 > L1 且 L2 > L3
```

#### 底分型（Bottom Fractal）

三根K线中，**中间K线的高点是三根中最低的，中间K线的低点也是三根中最低的**。

```
底分型示意：
 ██      ██
  \    /
   \  /
    ██  ← 第2根（最低）

条件：L2 < L1 且 L2 < L3，且 H2 < H1 且 H2 < H3
```

### 2.2 分型的严格定义（完整表述）

- **顶分型**：不含包含关系的3根K线，第2根K线的高点是三根中最高的，且第2根K线的低点也是三根中最高的
- **底分型**：不含包含关系的3根K线，第2根K线的高点是三根中最低的，且第2根K线的低点也是三根中最低的

> **关键**：分型的定义是在**处理完包含关系之后**的K线序列上进行的。

### 2.3 包含关系与K线合并

包含关系处理是缠论分析的第一步，也是最基础的预处理工作。

#### 包含关系的定义

两根相邻K线，如果一根K线的高低点完全包含另一根K线的高低点，就构成包含关系。

```
包含关系的两种情况：

情况A：前包后（第1根包含第2根）
  ┌───┐
  │ 1 │ ← 高点和低点范围更大
  │ ┌─┤
  │ │2│
  │ └─┤
  └───┘

情况B：后包前（第2根包含第1根）
  ┌─┐
  │1│
  │ └───┐
  │     │2│ ← 高点和低点范围更大
  └─────┘
```

**数学条件**：设第n根K线的高低点为(Hn, Ln)，第(n+1)根为(Hn+1, Ln+1)：

- (Hn ≥ Hn+1 且 Ln ≤ Ln+1) → 第n根包含第n+1根
- (Hn+1 ≥ Hn 且 Ln+1 ≤ Ln) → 第n+1根包含第n根

#### 合并规则（按方向）

合并规则取决于**序列的处理方向**：

**向上处理时**（正在处理上升序列）：

```
取 高者之高 + 低者之高
即：新高 = max(H1, H2)，新低 = max(L1, L2)
```

**向下处理时**（正在处理下降序列）：

```
取 高者之低 + 低者之低
即：新高 = min(H1, H2)，新低 = min(L1, L2)
```

#### 处理方向的确定

处理方向由**最近两根非包含K线**的关系决定：

- 如果第n根低点 < 第n+1根低点，且第n根高点 < 第n+1根高点 → **向上处理**
- 如果第n根低点 > 第n+1根低点，且第n根高点 > 第n+1根高点 → **向下处理**
- 如果高低点方向不一致（如n的高点 < n+1的高点，但n的低点 > n+1的低点），则看包含关系的具体形态，多数情况下该序列本身就存在包含关系，需先处理

#### 合并示例

```
原始K线序列（假设向下处理方向）：

K1: ████        H=100, L=90
K2:  ████       H=98, L=92  ← K1包含K2
K3:   ████      H=96, L=88

步骤1: K1和K2存在包含关系，方向为向下
       合并K1+K2 → 新K(100,92)：新高=min(100,98)=98，新低=min(90,92)=90
       不对，方向向下时：新高=min(H1,H2)，新低=min(L1,L2)
       合并后：H=min(100,98)=98, L=min(90,92)=90

步骤2: 合并后的K(98,90)与K3(96,88)
       K3高点96<98，K3低点88<90，非包含，可找分型
```

> **实战提示**：包含关系处理是缠论的**第一道工序**。许多初学者在这一步出错，导致后续的笔、线段全部错乱。务必仔细处理。

### 2.4 分型的延续与中继

分型出现后有两种命运：

| 类型 | 特征 | 意义 |
|------|------|------|
| **中继分型** | 很快被反向突破，趋势延续 | 原趋势继续 |
| **转折分型** | 没有被反向有效突破，成为真正的顶/底 | 趋势可能反转 |

判断中继还是转折的关键在于该分型是否被后续走势"破坏"——即分型之后是否形成了反向的笔。

## 3. 笔（Bi）

### 3.1 笔的定义

**笔（Bi）** 是缠论中连接相邻顶底分型的线段，是比K线高一阶的基本构件。

- **向上笔**：从底分型开始，到顶分型结束，期间不存在其他顶分型或底分型的干扰
- **向下笔**：从顶分型开始，到底分型结束，期间不存在其他顶分型或底分型的干扰

### 3.2 笔的构建规则

一条有效的笔必须同时满足：

1. **顶底分型交替**：顶分型和底分型必须交替出现，不能出现两个连续的顶分型或底分型
2. **至少5根独立K线**：顶分型和底分型之间（不含顶底分型自身）至少要有1根独立K线。加上两端的分型，总共至少5根独立K线（处理后）
3. **顶必须高于底**：向上笔中，顶分型的最高点必须高于底分型的最低点；向下笔中，顶分型的最高点必须高于底分型的最低点
4. **顶底分型不共用K线**：顶分型和底分型不能有K线重叠

```
向上笔结构（至少5根独立K线）：

     顶分型 ← 第5根
      /|\
     / | \
    /  |  \
   /   |   \
  /    |    \  ← 至少1根中间K线
 /     |     \
底分型  ← 第1根

条件：
- 底分型最低点 < 顶分型最高点
- 顶底分型之间至少1根K线不属于两端分型
- 整个序列无包含关系（已处理）
```

### 3.3 向上笔与向下笔

```
向上笔 (Up Bi)：
  - 起点：底分型的最低点
  - 终点：顶分型的最高点
  - 方向：从低到高
  - 符号：↗

向下笔 (Down Bi)：
  - 起点：顶分型的最高点
  - 终点：底分型的最低点
  - 方向：从高到低
  - 符号：↘
```

在一段完整的K线序列中，笔按照 **↗ ↘ ↗ ↘ ...** 的节奏交替出现。

### 3.4 笔的破坏

笔的破坏是判断线段终结的关键信号。

**笔被破坏**：一笔走出来之后，如果反向的一笔突破了该笔的终点（向上笔的顶被向下突破，或向下笔的底被向上突破），则该笔被破坏。

```
向上笔被向下笔破坏：

     ↖ 顶分型（笔的终点）
    / \
   /   \
  /     \        ← 向下笔突破此位置，
 /       ↘       即构成对该向上笔的破坏
/         \
           ↘ 新底低于前底？→ 笔被破坏
```

> **注意**："笔被破坏"与"线段被笔破坏"是两个不同层级的概念，后者见线段章节。

### 3.5 笔的常见误区

- **误区1**：认为只要看到顶分型和底分型就可以连成笔。实际上还需满足至少5根独立K线的条件
- **误区2**：在新的一笔还没有完全走出来之前就匆忙画笔。笔需要等反向分型确认后才能界定
- **误区3**：忽略了包含关系处理，导致分型识别错误

## 4. 线段（Segment）

### 4.1 线段的定义

**线段（Segment）** 是缠论中比笔更高一级的结构，由**至少三笔**构成，且起始三笔必须有重叠。

线段是缠论中最重要但也最容易混淆的概念之一。它是连接"笔"与"中枢"的中间层次。

```
线段的基本形态（至少3笔）：

向上线段：
     ↗ 第3笔
    / \
   /   \
  /     \  ← 重叠区间
 /       \
/    ↗    \
    /      第1笔
   /
  /
 ↗ 第2笔（回撤笔）

关键：第1、2、3笔必须有重叠部分（图中的重叠区间）
```

### 4.2 线段的严格规则

1. 线段至少由**3笔**构成（向上线段：上-下-上；向下线段：下-上-下）
2. 前三笔**必须有重叠区间**（即有一个共同的价位区间被三笔都触及）
3. 线段的方向与第一笔的方向相同
4. 线段内部不允许出现"笔的包含"（线段内各笔之间不存在包含关系）

### 4.3 线段被笔破坏的条件

这是缠论中最核心的判断之一。

**线段被笔破坏**：当且仅当——

- 对于**向上线段**：向下的一笔跌破该线段的最后一个向上笔的起点
- 对于**向下线段**：向上的一笔升破该线段的最后一个向下笔的起点

```
向上线段被破坏示意：

最后一笔向下的终点假设在第3笔的起点以下：

     ↗ 第5笔（最后一笔向上）
    / \
   /   \
  /     ↘ ← 
 /        第6笔（向下）跌破第5笔起点 → 线段被破坏
/    ↗     
    /
   /
  ↗ 第3笔起点

若第6笔的低点 < 第5笔的起点 → 线段被破坏
```

### 4.4 线段终结判断

线段终结的条件是**被反向线段破坏**：

- 原向上线段 + 一个向下线段（至少3笔且被确认）= 原线段终结
- 原向下线段 + 一个向上线段（至少3笔且被确认）= 原线段终结

**线段的"特征序列"判断法**是缠论中最精确的线段终结判断方法：

1. 将线段内的每一笔视为一个"特征元素"
2. 将这些特征元素按包含关系处理后
3. 在特征序列上寻找顶底分型
4. 当特征序列出现顶分型/底分型，且满足分型条件时，原线段在该分型位置终结

这是缠论中较为高阶的内容，初学者可先掌握基本的"三笔反向"判断法。

### 4.5 线段与笔的关系总结

```
层级结构：
  K线（含包含关系处理）
    ↓ 组成
  分型（顶分型/底分型）
    ↓ 连接（交替，≥5K线）
  笔（向上笔/向下笔）
    ↓ 组成（≥3笔，有重叠）
  线段
    ↓ 组成（≥3线段重叠）
  中枢
```

## 5. 中枢（Pivot/Zhongshu）

### 5.1 中枢的定义

**中枢（Zhongshu / Pivot）** 是缠论最核心、最独创的概念。它定义了走势的"价格重心"。

> **中枢定义**：某级别走势中，被**至少三个连续次级别走势类型**所**重叠**的价格区间。

```
中枢的几何定义：

取三段连续次级别走势的重叠区间：
  重叠上沿 = min(三个走势的高点)
  重叠下沿 = max(三个走势的低点)

示意图：
走势A高点 ────────────┐
                       │
走势B高点 ──────┐      │
                 │      │
     ┌───────────┴──────┴──┐
     │   中枢（重叠区间）     │
     └───────────┬──────┬──┘
                 │      │
走势C低点 ───────┘      │
                         │
走势A低点 ───────────────┘

中枢上沿（ZG）= min(A高, B高, C高)
中枢下沿（ZD）= max(A低, B低, C低)
中枢区间 = [ZD, ZG]
```

### 5.2 中枢的三种形态

| 形态 | 条件 | 含义 |
|------|------|------|
| **中枢新生** | 三个次级别走势重叠形成一个新的中枢，且不与前中枢重叠 | 产生新的价格重心，走势可能在持续 |
| **中枢延伸（Extension）** | 走势围绕中枢震荡，次级别走势不断回到中枢区间内 | 中枢级别尚未升级，盘整延续中 |
| **中枢扩展/扩张（Expansion）** | 两个同级别中枢的波动区间发生重叠，合并为更大级别中枢 | 级别升级，原走势类型死亡 |

### 5.3 中枢延伸

**中枢延伸**：中枢形成后，后续的次级别走势不断回到中枢区间（即触及[ZD, ZG]），导致中枢持续但级别不变。

```
中枢延伸示意：

     ┌──────┐
     │ 中枢 │ ← 后续走势反复回到此区间
     └──────┘
  ↗  ↘  ↗  ↘  ↗  ↘  ↗  ↘
 /    \/    \/    \/    \
```

当延伸达到**9段**次级别走势时，中枢自动升级为更高级别中枢。

### 5.4 中枢扩展（扩张）

**中枢扩展**：两个同级别中枢的**波动区间**发生重叠。

```
中枢扩展示意：

   中枢1                   中枢2
  ┌─────┐                ┌─────┐
  │     │    重叠区域     │     │
  └─────┘ ← ─ ─ ─ ─ ─ → └─────┘
     │                      │
      ↘                    ↗
        ↘                ↗
           ┌──────────┐
           │ 高级别中枢 │  ← 两个中枢合并升级
           └──────────┘
```

中枢扩展意味着**原走势类型结束**，市场进入更大级别的盘整。

> **区分**：中枢延伸 → 级别不变；中枢扩展 → 级别升级。

### 5.5 中枢的递归定义（级别视角）

中枢的定义是**递归**的：

- **1分钟级别中枢** = 3个1分钟线段的重叠区间
- **5分钟级别中枢** = 3个1分钟走势类型的重叠区间
- **30分钟级别中枢** = 3个5分钟走势类型的重叠区间
- ...以此类推

这意味着中枢天然具有**级别属性**，这也是缠论能够进行多级别联立分析的基础。

## 6. 走势类型

### 6.1 两种基本走势类型

缠论将所有走势分为两种基本类型：

#### 盘整（Consolidation）

> **盘整**：在任何级别的任何走势中，只包含**一个中枢**的走势类型。

```
盘整结构：
     ┌───────────┐
     │   中枢     │ ← 唯一的一个中枢
     └───────────┘
 ↗              ↘
/                 \
进入段              离开段
```

#### 趋势（Trend）

> **趋势**：在任何级别的任何走势中，包含**至少两个同向、不重叠的中枢**的走势类型。

```
上涨趋势（至少2个中枢）：
         ┌───────┐        ┌───────┐
         │ 中枢2  │        │ 中枢1  │
         └───────┘        └───────┘
 ↗      ↗       ↘      ↗       ↘
/      /          \    /          \
────进入段────连接段─────离开段────→

中枢之间不能有重叠（有重叠就是扩展了）
```

趋势分为：
- **上涨趋势**：中枢不断上移，每个中枢的低点都高于前一个中枢的高点
- **下跌趋势**：中枢不断下移，每个中枢的高点都低于前一个中枢的低点

### 6.2 走势终完美定理（走势分解定理）

这是缠论最核心的两大定理：

#### 走势分解定理一

> 任何级别的任何走势，都可以分解为同级别"盘整"、"下跌趋势"与"上涨趋势"三种走势类型的连接。

这意味着任何走势都可以被唯一地分解为这三种基本类型的序列。例如：

```
一段完整的走势 = 盘整 + 上涨趋势 + 盘整 + 下跌趋势 + 盘整
```

#### 走势分解定理二（"走势终完美"的等价表述）

> 任何级别的任何走势类型，都至少由三个次级别走势类型构成。

- 盘整 = 至少3段次级别（进入+中枢震荡+离开）
- 趋势 = 至少5段次级别（a + A + b + B + c，即进入段+中枢A+连接段+中枢B+离开段）

### 6.3 a+A+b+B+c 标准模型

缠论将趋势抽象为以下标准模型：

```
a + A + b + B + c

其中：
  a = 进入段（次级别走势）
  A = 第一个中枢（至少3个次级别走势重叠）
  b = 连接段（次级别走势，连接两个中枢）
  B = 第二个中枢（至少3个次级别走势重叠）
  c = 离开段（次级别走势，从中枢B离开）
```

```
示意：
      a ↗    A（中枢1）    b ↗    B（中枢2）    c ↗
     /   ┌───────────┐   /   ┌───────────┐   / \
    /    │  震荡区间   │  /    │  震荡区间   │  /   \
   /     └───────────┘ /     └───────────┘ /     \
  /                   /                   /       \
```

**背驰判断的关键**：比较 c段 与 b段（或 a段）的力度，当 c 的力度明显弱于 b（或 a）时，发生**趋势背驰**。

### 6.4 走势类型与中枢的关系

| 走势类型 | 中枢数量 | 中枢关系 |
|----------|----------|----------|
| 盘整 | 1个 | — |
| 上涨趋势 | ≥2个 | 不重叠，中枢依次上移 |
| 下跌趋势 | ≥2个 | 不重叠，中枢依次下移 |

## 7. 级别（Level）

### 7.1 级别的概念

**级别（Level）** 是缠论区别于所有其他技术分析理论的关键维度。大多数技术分析理论只看单一时间框架，而缠论通过级别的递归关系实现了**多级别联立分析**。

级别不是简单的"时间周期"（如日线、小时线），而是**走势类型自组织的层级结构**。

```
级别的比喻——俄罗斯套娃：

  ┌─────────────────────────────┐
  │       日线级别走势            │
  │  ┌───────────────────────┐  │
  │  │   30分钟级别走势        │  │
  │  │  ┌─────────────────┐  │  │
  │  │  │ 5分钟级别走势     │  │  │
  │  │  │  ┌───────────┐  │  │  │
  │  │  │  │ 1分钟级别  │  │  │  │
  │  │  │  └───────────┘  │  │  │
  │  │  └─────────────────┘  │  │
  │  └───────────────────────┘  │
  └─────────────────────────────┘

每一层都有自己独立的走势类型和买卖点
```

### 7.2 级别递归

级别递归是缠论技术体系中最精妙的部分：

```
级别递归规则：

本级别笔 ≈ 次次级别线段 ≈ 次次次级别走势类型

更具体的递推关系（以1分钟为最小级别）：

1分钟K线 → 1分钟笔 → 1分钟线段 → 1分钟走势类型
                                    ↓
                         5分钟笔 ← ─┘（约等于1分钟走势类型）
                                    ↓
                         5分钟线段 → 5分钟走势类型
                                        ↓
                             30分钟笔 ← ─┘
                                        ↓
                             30分钟线段 → 30分钟走势类型
                                            ↓
                                 日线笔 ← ──┘
```

**实际操作中的级别划分**（以加密货币/股票为例）：

| 级别 | 大致时间跨度 | 常用图表周期 |
|------|-------------|-------------|
| 1分钟级别 | 数十分钟～数小时 | 1分钟K线 |
| 5分钟级别 | 数小时～1-2天 | 5分钟K线 |
| 30分钟级别 | 1天～1周 | 30分钟K线 |
| 4小时级别 | 数天～数周 | 4小时K线 |
| 日线级别 | 数周～数月 | 日K线 |
| 周线级别 | 数月～数年 | 周K线 |

### 7.3 不同级别联立分析

> **缠论的核心操作原则**：大级别定方向，小级别找买卖点。

```
联立分析框架：

  周线级别：确定长期趋势方向（牛市/熊市/震荡）
      ↓
  日线级别：确定中期走势类型（盘整/趋势）、中枢位置
      ↓
  30分钟级别：确定短期走势结构，识别可能的买卖点区域
      ↓
  5分钟级别：精确定位买卖点（区间套）
      ↓
  1分钟级别：最精确的入场/离场时机
```

**操作逻辑**：

1. 在**大级别**上确认当前处于什么走势类型（上涨趋势、下跌趋势还是盘整）
2. 如果是上涨趋势，只在大级别买点区域用**小级别**找精确买点做多
3. 如果是下跌趋势，只在大级别卖点区域用**小级别**找精确卖点做空
4. 如果是盘整，可以高抛低吸（中枢震荡操作）

### 7.4 区间套原理

**区间套（Nesting）** 是缠论精确定位买卖点的终极工具。

> **区间套定理**：大级别的买卖点，必然对应着某个小级别的第一类买卖点。

```
区间套精确定位买点：

日线级别：出现第一类买点信号（趋势背驰）
    ↓ 缩小级别进入
30分钟级别：在日线买点区域，30分钟也出现趋势背驰
    ↓ 继续深入
5分钟级别：在30分钟买点区域，5分钟也出现趋势背驰
    ↓ 最终精确
1分钟级别：在5分钟买点区域，1分钟背驰点 = 最终精确入场点

这就像用不同倍数的放大镜逐层聚焦
```

**区间套操作原则**：

- 大级别未出现买点信号 → 小级别即使有买点也不操作（因为可能是假信号）
- 大级别出现买点信号 + 小级别确认 = **高胜率买点**
- 多个级别同时共振 → **最佳交易机会**

## 8. 背驰（Divergence）

### 8.1 背驰的概念

**背驰（Divergence）** 是缠论中判断走势力度衰竭的核心工具，也是三类买卖点判断的基础。

> **背驰的本质**：走势的价格在创新高（或新低），但内在的动能（力度）却在衰减，最终导致走势反转。

缠论的背驰与传统的MACD背离有本质区别：缠论的背驰是**结构性的**——它基于走势类型的完整结构来判断，而不仅仅是指标的背离。

### 8.2 趋势背驰 vs 盘整背驰

#### 趋势背驰

**趋势背驰**：发生在**趋势的最后一个中枢的离开段**。条件是c段相对于b段（或进入段a）力度明显减弱。

```
趋势背驰结构：

a ↗    A    b ↗    B    c ↗（背驰段）
/   ┌─────┐   /   ┌─────┐   /
    │中枢A│       │中枢B│    ↓ 力度不足，发生背驰
    └─────┘       └─────┘

判断：c段的力度 < b段的力度 → 趋势背驰
结果：趋势结束，转为盘整或反趋势
```

**趋势背驰后的三种演化**（"背驰后有三种情况"定理）：

1. **最后一个中枢的级别扩展**：背驰后的反弹触及中枢B的波动低点，但未回到中枢，只是震荡延伸，中枢级别由原级别扩展为大一级别
2. **更大级别的盘整**：背驰后反弹回到中枢B区间，走势转化为更大级别的盘整
3. **反趋势**：背驰后走势直接反转，形成与原趋势方向相反的新趋势（这是最强的一种情况）

#### 盘整背驰

**盘整背驰**：发生在**盘整中枢的离开段**。即离开段与进入段力度相比减弱。

```
盘整背驰结构：

    ┌─────────┐
    │  中枢    │
    └─────────┘
 ↗              ↘ 力度减弱
/                  \
进入段a             离开段b

判断：b段的力度 < a段的力度 → 盘整背驰
结果：盘整延续，走势被拉回中枢
```

> **关键区别**：趋势背驰意味着原趋势的终结；盘整背驰只意味着走势被拉回中枢继续震荡。

### 8.3 MACD辅助判断

缠论中，MACD是辅助判断背驰最常用的工具，但不是唯一工具。

#### 面积比较法

这是缠论原创的背驰判断方法，比简单的MACD背离更精确。

**方法**：在MACD指标图上，比较两段同向走势对应的**MACD红绿柱的面积之和**。

```
面积比较法示意（以趋势背驰为例）：

MACD柱状图：
  b段对应的面积           c段对应的面积
  ████████████           ████ ← 面积明显变小
  ████████████           ████
  ████████████           ████
  ████████████
  ████████████

当 c段对应的MACD面积 < b段对应的MACD面积 → 背驰确认
```

**面积计算要点**：

- 上涨趋势中，比较MACD红柱（正值）的面积
- 下跌趋势中，比较MACD绿柱（负值）的面积
- 柱体围绕零轴上下波动时，取绝对值累加

#### 黄白线（DIF/DEA）辅助判断

- **趋势背驰**：c段创新高/新低，但MACD黄白线不创新高/新低，甚至出现反向交叉
- **尤其注意**：c段对应的黄白线如果远远低于b段对应的黄白线高度，是极强的背驰信号

#### MACD判断背驰的局限性

> **重要提示**：MACD只是辅助工具。**结构上的背驰是第一位的，MACD是第二位的**。即使MACD没有明显背离，只要结构上c段力度明显小于b段（例如c段内部出现小级别背驰），也构成背驰。

### 8.4 力度比较的多种方法

除了MACD面积比较，缠论还提供了其他比较力度的方法：

1. **涨跌幅比较**：直接比较两段的涨跌幅度（百分比）
2. **角度（斜率）比较**：走势的斜率越陡，力度越强
3. **K线数量比较**：在相同时间内，K线数量越多（波动越频繁），力度越弱
4. **成交量比较**：放量上涨→力度强；缩量上涨→力度弱

### 8.5 背驰的判断流程图

```
判断背驰的流程：

1. 确认走势结构
   - 是否满足 a+A+b+B+c 趋势结构？
   - 还是只有一个中枢的盘整？

2. 确定比较对象
   - 趋势背驰：比较c段与b段的力度
   - 盘整背驰：比较离开段与进入段的力度

3. 力度比较
   - MACD面积比较（主方法）
   - 黄白线高度比较（辅助）
   - 涨跌幅/斜率比较（补充）

4. 区间套确认
   - 进入小级别查看c段内部是否也出现背驰
   - 如果c段内部背驰 + 大级别背驰 → 高度确认

5. 得出结论
   - 趋势背驰 → 趋势终结，准备反向操作
   - 盘整背驰 → 走势被拉回中枢，高抛低吸
```

## 9. 买卖点

### 9.1 三类买卖点概览

缠论定义了**三类买点和三类卖点**，它们是整个理论体系的最终输出——所有分析最终都指向买卖点的定位。

```
三类买点（以做多为例）：

        第1类买点        第3类买点
            ↓                ↓
    ────┐  ┌──────────┐  ┌────────
         \/            \/
         /\            /\
    ────┘  └──────────┘  └── 中枢 ──
            ↑
        第2类买点

第1类买点：趋势背驰后的最低点（抄底）
第2类买点：第1类买点后，回踩不破前低的次低点（确认）
第3类买点：突破中枢后，回踩不破中枢上沿的点（追涨）
```

### 9.2 第一类买卖点

> **第一类买点**：某级别下跌趋势背驰后的最低点。
> **第一类卖点**：某级别上涨趋势背驰后的最高点。

**特征**：

- 必须发生在**趋势的最后一个中枢的离开段背驰之后**
- 是**反转点**，意味着原趋势的终结
- 第一类买点之后，必然出现一个至少回到该趋势最后一个中枢区间的反弹
- 是所有买点中**盈亏比最高**的，但也是**最难精确判断**的

```
第一类买点结构：

           最后一个中枢
           ┌───────────┐
           │  中枢B     │
           └───────────┘
                      \
                       \ c段（背驰段）
                        \
                         \ 力度衰竭
                          ★ ← 第一类买点
                          ↑
                    趋势背驰确认
```

**操作要点**：

- 必须等背驰段（c段）的内部结构也完成（通过区间套）
- 可以在c段内部的小级别第一类买点处入场
- 止损设在该级别的最低点下方
- 目标：至少反弹到最后一个中枢的区间（中枢B的波动低点以上）

### 9.3 第二类买卖点

> **第二类买点**：第一类买点之后的次低点（不破前低）。
> **第二类卖点**：第一类卖点之后的次高点（不破前高）。

**特征**：

- 出现在第一类买卖点之后
- 是**确认性买点**——如果第一类买点没有错过，第二类买点是安全的加仓/确认位置
- 如果错过第一类买点，第二类买点是最好的补救机会
- 安全边际高，但盈亏比不如第一类

```
第二类买点结构：

           ┌───────────┐
           │  中枢B     │
           └───────────┘
                      \
  第一类买点 ★          \
              \          \
               \    ↗    \
                \  /      \
                 \/        ↘  第二类买点 ★
                  ↘         ↗ （回踩不破前低）
```

**特殊情形**：在非常强势的反转中，第二类买点可能与第一类买点重合（即反弹后直接一路上涨，没有明显的二次回踩），此时以第一类买点为准。

### 9.4 第三类买卖点

> **第三类买点**：突破中枢上沿后，回踩不破中枢上沿（ZG）的点。
> **第三类卖点**：跌破中枢下沿后，反抽不破中枢下沿（ZD）的点。

**特征**：

- 发生在**中枢被有效突破之后**
- 确认中枢突破的有效性
- 是最安全的**趋势延续型买点**
- 盈亏比最低，但胜率最高

```
第三类买点结构：

                     ↗ 回踩不破中枢上沿
                    /      ★ ← 第三类买点
                   /
    ┌───────────┐ /
    │   中枢     │/
    └───────────┘
                 \
                  \
                   ↘ 突破中枢！

条件：离开中枢后的回踩，低点 > 中枢上沿（ZG）
```

> **重要**：第三类买点出现后，该走势至少会产生一个新的同级别中枢（即原来的中枢被确认后，走势延续为趋势）。

### 9.5 买卖点与级别的对应关系

缠论买卖点的一个核心洞见是：**买卖点天然与级别绑定**。

| 买卖点类型 | 出现的走势环境 | 对应级别 | 操作周期 |
|-----------|---------------|----------|---------|
| 第一类买点 | 趋势背驰 | 本级别趋势终结 | 长线/波段 |
| 第二类买点 | 背驰后回踩确认 | 本级别趋势反转确认 | 中线 |
| 第三类买点 | 中枢突破回踩 | 本级别中枢+次级别回踩 | 短线/中线 |

### 9.6 区间套精确定位买卖点

这是缠论操作中最实用的技术：

```
以第一类买点为例的区间套定位：

Step 1: 日线级别确认
  → 日线下跌趋势完成 a+A+b+B+c 结构
  → c段力度明显弱于b段（MACD面积比较）
  → 日线级别第一类买点区域确认

Step 2: 30分钟级别细化
  → 进入30分钟图，在c段内部找结构
  → 30分钟c段内部出现小级别趋势背驰
  → 30分钟第一类买点确认

Step 3: 5分钟级别精确
  → 在30分钟买点区域，切换到5分钟图
  → 5分钟出现底背驰
  → 精确入场点确认

Step 4: 入场
  → 在5分钟背驰点入场做多
  → 止损设在日线级别最低点下方
  → 第一目标：日线最后一个中枢的波动低点
```

### 9.7 买卖点操作原则

**三大纪律**：

1. **顺势而为**：大级别上涨趋势中只做多（用买点），大级别下跌趋势中只做空（用卖点）
2. **级别对应**：什么级别看到买卖点，就什么级别操作，不要用日线买点做1分钟短线
3. **止损必设**：任何买点都不是100%，背驰之后还有背驰段延伸的可能

## 10. 实战应用

### 10.1 缠论在BTC-USDT永续合约中的应用

加密货币市场（尤其是BTC-USDT永续合约）具有**7×24小时连续交易、高波动性、高流动性**的特点，缠论在此类市场中具有天然优势。

#### 永续合约的特殊性

| 特点 | 对缠论的影响 |
|------|-------------|
| 无收盘概念 | K线连续，分型识别不受跳空影响（优势） |
| 资金费率 | 长期持仓成本需考虑，影响中枢延伸判断 |
| 高杠杆 | 需要更精确的入场点（区间套尤为重要） |
| 高波动 | 笔和线段的破坏更频繁，需提升分析级别 |
| 7×24交易 | 级别对应关系与传统股市不同 |

#### 推荐的分析级别组合（BTC永续合约）

```
主操作级别组合：

做短线（持仓数小时～1天）：
  主级别：15分钟或30分钟
  辅助级别：1小时（定方向）
  精确级别：5分钟（区间套定位）

做中线（持仓数天～1周）：
  主级别：1小时或4小时
  辅助级别：日线（定方向）
  精确级别：15分钟（区间套定位）

做长线（持仓数周～数月）：
  主级别：日线
  辅助级别：周线（定方向）
  精确级别：4小时（区间套定位）
```

### 10.2 常见误区

#### 误区一：级别混乱

**表现**：在1分钟图上找日线级别的买卖点，或用日线中枢的位置直接在5分钟图上操作。

**纠正**：严格执行级别对应——本级别的中枢必须用本级别的图来识别，买卖点也必须在本级别确认后再用次级别精确。

#### 误区二：过度交易

**表现**：每个小级别的分型都想操作，忽略了只有三类买卖点才是有效的操作点。

**纠正**：严格只在三类买卖点处操作，中枢震荡中的普通波动应该忽略。

#### 误区三：背驰后一定反转

**表现**：看到背驰就重仓杀入，结果背驰段延伸，继续跌/涨。

**纠正**：背驰是力度衰竭的信号，但不等于立即反转。背驰之后可能出现"背了又背"（背驰段延伸），应：
- 等待背驰段内部的小级别背驰确认
- 分批建仓而非一次性满仓
- 严格设置止损

#### 误区四：忽略包含关系处理

**表现**：不处理包含关系就直接画分型和笔，导致整个分析体系错乱。

**纠正**：包含关系处理是缠论分析的第一步，必须严格逐根处理。宁可多花时间确保基础结构正确。

#### 误区五：画线的主观性

**表现**：不同人画出的笔和线段不一样，导致分析结果不同。

**纠正**：缠论的笔和线段规则是严格确定的（只要严格按照规则操作）。差异来自于对包含关系处理、分型确认的不同理解。应统一标准、反复核对。

### 10.3 结合成交量分析

缠论原著中对成交量的论述不多，但在实战中，成交量是验证背驰和买卖点的重要辅助工具。

#### 量价配合原则

| 情况 | 量价关系 | 含义 |
|------|---------|------|
| 上涨放量 | 价格↑ + 成交量↑ | 上涨有力，趋势健康 |
| 上涨缩量 | 价格↑ + 成交量↓ | 上涨乏力，可能背驰 |
| 下跌放量 | 价格↓ + 成交量↑ | 下跌加速，恐慌出逃 |
| 下跌缩量 | 价格↓ + 成交量↓ | 下跌动能衰竭，可能见底 |
| 突破放量 | 突破中枢 + 成交量大增 | 有效突破，第三类买卖点确认 |

#### 成交量+缠论结合技巧

- **背驰确认**：c段不仅MACD面积变小，同时成交量也明显萎缩 → 背驰信号更可靠
- **第三类买点确认**：突破中枢上沿时放量，回踩时缩量 → 完美的第三类买点结构
- **中枢突破真伪**：无量突破中枢 → 大概率是假突破；放量突破 → 大概率有效

### 10.4 结合其他技术指标

#### MACD（深度结合）

缠论与MACD的结合最为紧密，除了前述的面积比较法外：

```
MACD多周期配合：

日线MACD方向    30分钟MACD方向    操作建议
───────────────────────────────────────────
   向上             向上          强力做多
   向上             向下          短空长多，等30分钟金叉
   向下             向下          强力做空
   向下             向上          短多长空，等30分钟死叉
   零轴附近          —            盘整，高抛低吸
```

> **零轴的意义**：MACD黄白线在零轴上方运行→多头主导；零轴下方→空头主导。中枢震荡中MACD围绕零轴反复穿越。

#### RSI（相对强弱指标）

```
RSI与缠论配合：

1. RSI背离确认背驰：
   → 价格创新高，RSI不创新高（顶背离）
   → 与缠论的趋势背驰对应，双重确认

2. RSI超买超卖与买卖点：
   → RSI > 70 区域 + 缠论第一类卖点信号 → 强烈卖出信号
   → RSI < 30 区域 + 缠论第一类买点信号 → 强烈买入信号

3. RSI在中枢震荡中的应用：
   → 中枢下沿 + RSI < 30 → 买入
   → 中枢上沿 + RSI > 70 → 卖出
```

#### 布林带（Bollinger Bands）

```
布林带与缠论配合：

1. 布林带收窄（缩口）：
   → 对应缠论的中枢延伸末期
   → 预示着即将突破，应密切关注第三类买卖点

2. 布林带开口：
   → 对应缠论的离开段（b段或c段）
   → 开口方向就是离开段方向

3. 价格沿布林带上轨/下轨运行：
   → 沿上轨运行：强势上涨，可能形成趋势
   → 沿下轨运行：强势下跌，可能形成趋势
   → 沿中轨震荡：中枢延伸中

4. 布林带与买卖点：
   → 价格触及下轨 + 底分型 → 潜在的买点区域
   → 价格触及上轨 + 顶分型 → 潜在的卖点区域
```

### 10.5 缠论在BTC交易中的综合实战案例

以下是一个典型的BTC-USDT永续合约缠论操作完整案例（模拟逻辑）：

```
标的：BTC-USDT 永续合约
时间框架：4小时主级别 + 1小时辅助 + 15分钟精确

【第一阶段：方向确认（4小时图）】
  - 识别走势类型：下跌趋势 a(跌) + A(中枢) + b(跌) + B(中枢) + c(跌)
  - 中枢A和B不重叠，确认下跌趋势
  - c段MACD面积明显小于b段 → 潜在趋势背驰
  - 结论：4小时级别可能的第一类买点正在形成

【第二阶段：中枢定位（1小时图）】
  - 在1小时图上确认中枢B的精确区间：ZD=28500, ZG=29200
  - c段内部在1小时图上呈现小级别下跌趋势
  - c段最后一个1小时中枢：ZD=27200, ZG=27500

【第三阶段：区间套精确定位（15分钟图）】
  - 在15分钟图上观察c段最后一个1小时中枢的离开段
  - 出现15分钟级别底背驰
  - 15分钟底分型形成，确认精确入场点：27120

【第四阶段：入场与风险管理】
  - 入场价位：27150（在15分钟底分型确认后入场）
  - 止损：26900（在c段最低点下方，约1%止损）
  - 仓位：总资金的5%（考虑杠杆后的风险敞口）
  - 第一目标：27500（中枢B的波动低点，约1.3%收益）
  - 第二目标：28500-29200（中枢B区间）
  - 第三目标：中枢A附近（如果反转力度强）

【第五阶段：持仓管理】
  - 价格到达27500后，移动止损至成本价
  - 价格到达28500后，移动止损至27800
  - 观察是否形成第二类/第三类买点，决定是否加仓
  - 如果1小时出现顶背驰，分批止盈离场
```

### 10.6 学习缠论的建议路径

```
推荐的缠论学习路径（由浅入深）：

  → 彻底掌握包含关系处理
  → 熟练识别顶底分型

  → 理解线段的概念和画法
  → 掌握中枢的定义和识别
  → 区分盘整和趋势

第三阶段（4-8周）：进阶篇
  → 理解级别递归
  → 掌握背驰判断（MACD面积比较法）
  → 学会识别三类买卖点

第四阶段（8周+）：实战篇
  → 多级别联立分析
  → 区间套精确定位
  → 大量复盘和模拟训练
  → 小资金实盘检验

关键原则：
  1. 每一步都要画图，不要只在脑子里想
  2. 大量复盘历史走势，验证自己的判断
  3. 先用模拟盘或极小仓位实盘
  4. 缠论是"不测而测"，不要用缠论去预测
  5. 纪律比技术更重要
```
