# 社交动态模板 v5.1

> 从 `analysis_template.py --social` 草稿增强的社交动态文案模板。
> 数字从脚本输出抄，表达用说人话的方式。

## 输出格式

```
{一句话炸裂标题} 🔥

📋 上条复盘
{上条时间} | BTC {方向} {入场}→{现价} ({change_pct:+.1f}%) {✅/❌/🟡}
           | ETH {方向} {入场}→{现价} ({change_pct:+.1f}%) {✅/❌/🟡}

🕐 BJ {月-日} {时:分} | BTC ${价格} | ETH ${价格} | FG:{数值}

💡 大资金在干嘛？
{威科夫阶段大白话 + VP POC位置}

📐 盘面看了什么？
{道氏方向 + 技术指标 + 共振标签}

📊 钱堆在哪？
{费率 + VP POC/VAH/VAL + 解读}

🌍 宏观给不给面子？
{FG + DXY/VIX/10Y + 关键事件}

---
{一锤定音的结论}

📍 BTC 支撑 {S1}→{S2} | 阻力 {R1}→{R2}
📍 ETH 支撑 {S1}→{S2} | 阻力 {R1}→{R2}

🎯 BTC {方向} | 入场{区间} | 止损{sl} | 止盈{tp} | RR 1:{rr}
🎯 ETH {方向} | 入场{区间} | 止损{sl} | 止盈{tp} | RR 1:{rr}
```

## 写作原则

| # | 规则 |
|---|------|
| 1 | 标题决定50%阅读量 — 数据震撼/反直觉/结论先行/悬念/挑衅五选一 |
| 2 | 开头2行定生死 — 反差、紧迫感、场景化 |
| 3 | 每段=解决一个问题 — 💡大资金/📐盘面/📊钱堆/🌍宏观 |
| 4 | 说人话不念经 — "Spring后Test成功" → "庄家砸到60638再回踩61060没破" |
| 5 | 价位用📍行内标注 — 比表格快扫 |
| 6 | 结论一句话说清方向+条件 |
| 7 | 收尾要让人记住 — 金句/数字/反差异，取其二 |
| 8 | 仓位一行写完 — 方向+入场+止损+止盈+RR，RR<1.5标⚠️ |
| 9 | 去废词 — 删"建议/可能/或许" |
| 10 | Telegram不发第二遍 |
| 11 | 每段≥2-3个精确数字 — 不要只说"放量"要说"量2.8M是正常3-5倍" |

## 强制步骤

1. 复盘上条 → `python3 scripts/review_last_post.py --save`
2. 写文案 → 从脚本输出抄数字
3. 核验 → `python3 verify_social_post.py "文案"`
4. 保存 → `python3 scripts/save_social_post.py`
5. 验证 → 确认 Post #N saved + text_len > 0

## ⚠️ 禁止

- 跳过核验
- 凭印象写数字（天数/期数/百分比必须从脚本输出逐条抄）
- FOMC/CPI前48h给方向建议
- B13结构止损时硬凑RR>1.0
